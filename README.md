# AIntropy Blog

This is where all AIntropy blog posts are written and reviewed before going live on aintropy.ai/blog.

---

## Workflow

1. Write post here as an HTML file
2. Preview locally: open the file via `file:///path/to/file.html` in your browser
3. Share with team for review
4. Once approved by Kunal, copy to Demo_walkthrough repo and push to go live
